# Anime-chat
A simple chat written in python for anime ai

## Features
- Character selection (pre-built and custom)
- Chat interface
- Custom character creation
- Template-based responses

## How to Run
Run: `python anime.py`

## Similar to TavernAI
This app provides a console interface for chatting with customizable AI characters, inspired by TavernAI but focused on simple template-based responses.
